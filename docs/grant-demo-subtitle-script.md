# RGS Grant Demo — Subtitle-First Recording Plan

Purpose: a **silent screen walkthrough** with subtitles added in post.
No voiceover required. Voiceover can be added later if desired.
Target length: **7–8 minutes**.

Tone: calm, trust-first, plain English. Short captions, 1–2 lines each.

Companion files:
- `docs/grant-demo-recording-runbook.md` — full recording runbook
- `docs/grant-demo-subtitles-draft.srt` — drop-in SRT file

---

## How to use this plan

1. **Record silent first.** Open the tabs in order. Move calmly. Don't rush.
2. **Hold each scene long enough** for the matching caption to stay on screen comfortably (3–6 seconds per caption).
3. **Add the SRT in post.** Most editors (iMovie, CapCut, DaVinci Resolve, Premiere) accept `.srt` directly.
4. **Sanity-check timing** against your actual take. Nudge caption start times if you held a scene longer.
5. **No voiceover needed.** If you add voice later, the captions still work as a baseline track.

## Caption style guide

- **Length**: max ~60 characters per line, max 2 lines.
- **Duration**: 3–6 seconds per caption. Never under 1 second.
- **Tone**: calm, declarative, no exclamation marks.
- **Punctuation**: minimal. No semicolons. Em-dashes are fine.
- **Numbers**: always digits (62, 72%, 0–1,000).
- **Brand names**: "RGS", "Revenue and Growth Systems", "QuickBooks". Keep capitalization exact.
- **Forbidden words**: proven, guaranteed, case study, real client outcome, AI knows.
- **Forbidden topics**: MMJ, cannabis, retail, SaaS, agencies, creators, roadmap expansion, industries outside trade and field service.
