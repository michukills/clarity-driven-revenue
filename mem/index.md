# Project Memory

## Core
Locked to dark theme: Background #1F1F1F, Primary #6B7B3A, Accent #8FA35A, Text #F5F5F5.
Fonts: Source Serif 4 (headings), Inter (body). Never serif for body.
Hosted at clarity-driven-revenue.lovable.app. Must remain on this domain.
Avoid KPI terminology. Use "Revenue Clarity" or "Structured Tracking" instead.
Auto-scroll to top on every route change. No anchor links.
No generic startup vibes, "hype" marketing language, stock or AI-generated imagery.
Third-party brand names MUST use exact official form: QuickBooks, Xero, FreshBooks, Stripe, Square, PayPal, HubSpot, Salesforce, Pipedrive, Google Analytics, Google Search Console, Meta Ads, Paycom, ADP, Gusto, Jobber, Housecall Pro, ServiceTitan. Never imply partnership/endorsement.
RGS/Matt voice: calm, direct, owner-respecting, system-focused. Never motivational/agency/AI-sounding. Banned: "unlock", "empower", "optimize", "scalable", "actionable insights", "game-changing", "next level", "growth hacks", "dominate", "crush it". Use guided independence — Revenue Control System™ is continued visibility, not dependency.
